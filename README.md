# little-bee-socket
This is a C++ TCP framework.

# Build and Test
## Command
```
> mkdir build
> cd build
> cmake ..
> make
> make doxygen
> ./bin/unit_test
> ./bin/example_simple
```

## Script
### Build all targets including tests and examples
```
> build.sh
```
### Build documentation
```
> build.sh doc
```
### Install library and header files of the project
```
> build.sh install
```
